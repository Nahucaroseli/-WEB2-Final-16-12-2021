Pregunta A
Una API REST es una interfaz para conectar sistemas que 
esten basados en protoloco HTTP y sirve para obtener y realizar funciones y datos, esos datos los devuelve en 
formatos como por ejemplo JSON y XML.

Algunas de las ventajas que tiene el utilizar una API REST es la experiencia del usuario,
 requiere menos recursos del servidor, tiene mucha escalabilidad y flexibilidad
